Virtual keyboard for typing Modern Hebrew and transliterated Phoenician Script / Paleo-Hebrew.

To give an example of how this Modern-Paleo Hebrew transliteration works, the Modern Hebrew phrase "הכתב העברי הקדום" or "The Ancient Hebrew Script" would be mapped to Paleo-Hebrew like this:

הכתב (HaKetav) → 𐤄𐤊𐤕𐤁 (HKTB)
העבר (HaIvri) → 𐤄𐤏𐤁𐤓𐤉 (H'BRY)
הקדום (HaKadum) → 𐤄𐤒𐤃𐤅𐤌 (HQDWM)

𐤄𐤊𐤕𐤁 𐤄𐤏𐤁𐤓𐤉 𐤄𐤒𐤃𐤅𐤌 → הכתב העברי הקדום
